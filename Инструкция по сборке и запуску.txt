Сборка:
для сборки контейнера используйте команду:

	docker build -t {имя_образа}:latest .

	при  сборке можно явно задать любой из параметров окружения:
		POSTGRES_PASSWORD
		POSTGRES_USER
		POSTGRES_DB
	Значения по умолчанию:
		POSTGRES_PASSWORD=wellcome
		POSTGRES_USER=user
		POSTGRES_DB=database
	вид команды сборки для именования параметров:
		
		docker build -t {имя_образа}:latest --build-arg {ПАРМЕТР}={ЗНАЧЕНИЕ} .
